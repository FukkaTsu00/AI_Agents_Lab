#  AI Agents – TP

##  AIDATA G2 – BAKKAR Alaaeddine
##  Project Overview

This project is part of the **AI Agents (TP)** practical work.

The objective is to experiment with multiple Large Language Model (LLM) providers and understand how AI agents can interact with different backends.

The project integrates:

*  OpenAI API
*  Groq API
*  Ollama (local LLM)
*  Postman for API testing

---

## 🛠️ Tech Stack

* Python 
* Virtual Environment (venv)
* OpenAI API
* Groq API
* Ollama 
* Postman
* Jupyter Notebook
